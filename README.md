## Hướng Dẫn Chạy Cục Bộ

### Yêu Cầu

Máy có cài Node.js

### Bước 1: Cài Đặt Dependencies

Mở terminal và chạy lệnh sau để cài đặt các dependencies:

```bash
npm install
```

### Bước 2: Khởi Chạy Dự Án

Sử dụng lệnh sau để bắt đầu dự án cục bộ:

```bash
npm start
```
